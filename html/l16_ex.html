Flexbox - what is that?

Flexbox is a CSS layout model that makes it easier to arrange items in a row or column. It helps elements adjust their size and position based on the space available.

Common Flexbox properties include flex-direction, justify-content, and align-items. These let you control alignment, spacing, and the overall layout of items in a container.
